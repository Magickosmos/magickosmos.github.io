Vision Project (Pharma Vision LLC)

Project works with transferring BMP files through various different color filters and kernels to try and speed up the process of defect detection. Various functions are present which make this easier such as resizing and kernel generation. Alongside other quality of life changes such as zooming in on cursor and image deletion. Below is a video demo of the project at its current state with permission from Pharma Vision LLC.

As seen in the video you first are able to select a folder of BMP images to open or a singular image. When the folder is open there is nothing previous so a brief error is thrown if you try to go back when nothing is previous. Otherwise pressing next brings up the next image and once again throws an error if there are no remaining bmp images. Other functionalities demonstrated in the demo include resizing images, deleting filters, changing the order and number of kernels applied, and applying filters to different images.
